# jquery-values

jQuery plugin to get / set values of multiple form elements at once.

```
// get values
jQuery('.form-selector').values();

// set values
jQuery('.form-selector').values({
	username:'default',
	email: 'email@domain.com',
	first_name: 'John',
	last_name: 'Doe'
});
```
## Installation

```
bower install jquery-values
```
